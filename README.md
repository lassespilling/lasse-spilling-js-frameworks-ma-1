# GETTING STARTED
#
###  Installing node modules
```
npm -i
```

###  Use npm script to run Parcel watch
```
"npm run dev"
```

###  Run npm script for Parcel build for production
This outputs to "dist" folder.
```
"npm run prod"
```

npm scripts can be edited in package.json
